

# Webscraper Bitcoin
Webscraper Bitcoin is a command-line application that retrieves the current price of Bitcoin from the Okx exchange website. It then prints the price in a human-readable format.

## Getting Started
To use the application, follow these steps:

Make sure you have Rust installed on your system.

Clone the repository to your local machine:



    git clone https://github.com/kakamorsch/webscraper_bitcoin.git

Navigate to the project directory:

    cd webscraper_bitcoin

Build the application:

cargo build --release
Run the application:

    ./target/release/webscraper_bitcoin

This will print the current price of Bitcoin in USD.

Usage
The Webscraper Bitcoin can be run using the following command:


    ./target/release/webscraper_bitcoin

It will retrieve the current price of Bitcoin from the Okx exchange website and print it in a human-readable format.
